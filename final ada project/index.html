Dijkstra's Algorithm With GUI




How To Use

  • Switch On Set vertex to make a vertice, click anywhere in the canvas to create a vertex.
  • To connect vertices switch on Draw edge then click on the vertices you want to connect.
  • To give weight of edge switch on Set Weight
  • To select the starting point switch on Start Point then click on the starting vertex to get the results.
